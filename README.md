Portal inmobiliario inspirado en el universo de GTA Vice City.
Desarrollado para una prueba técnica.

Enlace [https://vercettilaravel-production.up.railway.app/login]

Tecnologías: Laravel, Eloquent ORM, Blade, MySQL, DomPDF

Funcionalidades

Sistema de autenticación con login y sesiones
Listado público de propiedades 
Panel de administración protegido por autenticación
CRUD completo de propiedades (crear, editar, eliminar)
Exportación del listado de propiedades a PDF
Persistencia de datos en base de datos MySQL
